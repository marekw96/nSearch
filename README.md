# nSearch
Simple TUI (terminal user interface) adapter for grep command with preview feature.

## How to 
*	`git clone --recursive https://github.com/marekw96/nSearch`
* `cd nSearch`
* `mkdir build`
* `cd build`
* `cmake ..`
* `make`
* ` ./nSearch`
